Botones

Los botones son elementos interactivos fundamentales en una interfaz.
Se utilizan para iniciar acciones, enviar formularios o navegar entre páginas.
En esta guía de estilos te mostramos los 3 tipos principales:

  1. Este es un botón primario.
    Usado para acciones principales/importantes.

  2. Este es un botón secundario.
    Usado para acciones secundarias/alternativas.

  3. Este es un botón terciario.
    Usado para acciones menos importantes.

Código (HTML)

                    <button class="btn btn--primary">Primario</button>
                    <button class="btn btn--secondary">Secundario</button>
                    <button class="btn btn--outline">Tercerario</button>

Código (CSS)

                       .btn {
                            font-family: Arial, sans-serif;
                            padding: 10px 18px;
                            border-radius: 6px;
                            border: none;
                            cursor: pointer;
                            transition: .2s ease;
                            font-size: 1rem;
                            margin: 5px;
                        }

                        .btn-primary {
                            background-color: #49708a;
                            color: #fff;
                        }

                        .btn-secondary {
                            background-color: #88abc2;
                            color: #fff;
                        }

                        .btn-tertiary {
                            background-color: transparent;
                            color: #010937;
                            border: 2px solid #d0e0eb;
                        }

Inputs

Los elementos <input> en HTML se usan para crear campos
campos interactivos en un formulario web, donde permite que los
usuarios ingresen y envien informacion de diferentes tipos, como texto, números,
correos electrónicos, etc.
Ejemplos de uso:

  1. Input de Texto :

    2. Los elementos <input> de tipo text crean campos de texto básicos de una sola línea.

    Código en HTML:

    <input type="text" id="text" name="text" placeholder="Ingrese su texto aquí">

  2. Input de Email :

    3. Los elementos <input> de tipo email crean campos para ingresar direcciones de correo electrónico.

    Código en HTML:

    <input type="email" id="email" name="email" placeholder="Ingrese su correo electrónico">

  3. Input de Password :

    4. Los elementos <input> de tipo password crean campos para ingresar contraseñas de forma oculta.

    Código en HTML:

    <input type="password" id="password" name="password" placeholder="Ingrese su contraseña">

  4. Input de Textarea :

    5. Los elementos <textarea> se utilizan para crear campos de texto de varias líneas.

    Código en HTML:

    <textarea id="textarea" name="textarea" rows="4" cols="50" placeholder="Ingrese su texto aquí"></textarea>

Formularios

Los formularios en HTML se utilizan para recopilar información de los usuarios. Estos permiten enviar datos como texto, correos, contraseñas u opciones seleccionadas a un servidor o procesarlos en la misma página.
Ejemplos de uso: registro de usuarios, inicio de sesión, encuestas y formularios de contacto.

  1. Formulario básico con estructura:










    2. Este ejemplo muestra un formulario con campos básicos para ingresar nombre, correo y mensaje.

    Código en HTML:

    <form>
      <label for="nombre">Nombre:</label>
      <input type="text" id="nombre" name="nombre" placeholder="Ingrese su nombre">

      <label for="correo">Correo electrónico:</label>
      <input type="email" id="correo" name="correo" placeholder="ejemplo@correo.com">

      <label for="mensaje">Mensaje:</label>
      <textarea id="mensaje" name="mensaje"></textarea>

      <button type="submit">Enviar</button>
    </form>

  2. Validación visual de formularios:










    3. En este ejemplo se aplican reglas de validación visual:
    - El campo Usuario requiere mínimo 4 caracteres.
    - El campo Correo electrónico debe tener un formato válido.
    - La Contraseña requiere al menos 6 caracteres.
    Si se intenta enviar el formulario sin cumplir los requisitos, el navegador muestra mensajes de error.

    Código en HTML:

    <form>
      <input type="text" id="usuario" name="usuario" required minlength="4">
      <input type="email" id="email2" name="email2" required>
      <input type="password" id="password2" name="password2" required minlength="6">
    </form>

Labels (Etiquetas de texto)

Los labels o etiquetas de texto con diferentes estilos y tamaños son fundamentales en el diseño porque permiten
establecer una jerarquía visual clara dentro de la interfaz.
Al utilizarlos correctamente se logra:

  1. Organización del contenido: Los títulos, subtítulos y textos informativos ayudan a estructurar la información de manera ordenada,
    permitiendo al usuario identificar lo más importante y lo secundario.
  2. Consistencia en el diseño: Definir estilos específicos evita que cada página use tamaños o colores
    distintos, manteniendo una identidad visual coherente en todo el proyecto.
  3. Mejor legibilidad y experiencia de usuario: Los diferentes tamaños y pesos tipográficos guían la lectura
    de forma natural, haciendo que la información sea más fácil de comprender.
  4. Semántica y accesibilidad: Al emplear etiquetas HTML como <h1>, <h2>, <h3> y <p>, se mejora la
    interpretación de la página tanto para los usuarios como para los buscadores, beneficiando la accesibilidad y el SEO.

  1. Título Principal:

    2. Título Principal (H1)

    Uso: Se utiliza una sola vez por página para indicar el tema principal o el encabezado más importante.
    Características: Fuente más grande, en negrita y llamativa.
    Ejemplo de aplicación: Nombre del sitio, título de una sección principal.

  2. Subtítulo:

    3. Subtítulo (H2)

    Uso: Se emplea para dividir el contenido en secciones secundarias.
    Características: Un poco más pequeño que el H1, pero aún con peso visual.
    Ejemplo de aplicación: Encabezados de apartados dentro de una página o sección.

  3. Subtítulo más pequeño:

    4. Subtítulo más pequeño (H3)

    Uso: Divide aún más las secciones, funcionando como subtítulos de los H2.
    Características: Tamaño intermedio, puede ser menos llamativo que el H2.
    Ejemplo de aplicación: Títulos de subsecciones o temas dentro de un bloque de texto.

  4. Texto descriptivo (párrafo - <p>):

    5. Texto descriptivo normal (párrafo)

    Uso: Sirve para mostrar el contenido principal en forma de párrafos.
    Características: Fuente estándar, tamaño legible y sin mucho peso visual.
    Ejemplo de aplicación: Explicaciones, descripciones, información detallada.

  5. Etiqueta pequeña (span - información secundaria):

    6. Etiqueta pequeña (span)

    Uso: Para resaltar fragmentos de texto dentro de un párrafo o mostrar información complementaria.
    Características: Fuente más pequeña, a veces en cursiva o con color distinto.
    Ejemplo de aplicación: Notas, aclaraciones, etiquetas de estado como "Nuevo", "Oferta", etc.

Código en HTML: 

                    <h1 class="label-title">Título Principal (H1)</h1>
                    <h2 class="label-subtitle">Subtítulo (H2)</h2>
                    <h3 class="label-small-subtitle">Subtítulo más pequeño (H3)</h3>
                    <p class="label-text">Texto descriptivo normal (párrafo)</p>
                    <span class="label-info">Etiqueta pequeña (span)</span>

Código en CSS: 

                    .label-title {
                        font-size: 2.5rem;
                        font-weight: bold;
                        color: #0a4b75;
                        }

                        .label-subtitle {
                        font-size: 2rem;
                        font-weight: 600;
                        color: #1e6495;
                        }

                        .label-small-subtitle {
                        font-size: 1.5rem;
                        font-weight: 500;
                        color: #5191c1;
                        }

                        .label-text {
                        font-size: 1rem;
                        font-weight: normal;
                        color: #2b879e;
                        }

                        .label-info {
                        font-size: 0.875rem;
                        font-style: italic;
                        color: #013750;
                        }

Links

Los links (enlaces) en HTML se crean con la etiqueta <a> y se pueden personalizar con
CSS para mostrar diferentes estados visuales.
Ejemplos:

  1. Link Normal (a):

    2. Es el estilo por defecto del enlace antes de que el usuario interactúe.

  2. Hover (a:hover):

    3. Se activa cuando el usuario pasa el cursor por encima del enlace.
    Se suele usar para dar una pista visual (cambiar color, subrayar, aumentar tamaño).

  3. Active (a:active):

    4. Se activa cuando el usuario hace clic en el enlace y lo mantiene presionado.
    Suele usarse para dar un efecto de "click real".

TOCA ESTE LINK

Código en HTML: 

                    <a href="https://css-tricks.com/guides/">TOCA ESTE LINK</a>

Código en CSS: 

                    /*Normal*/
                    a {
                        color: #101652;
                        text-decoration: none;
                        font-size: 18px;
                        transition: .2s ease;
                    }
                    /*HOVER*/
                    a:hover {
                         color: #49708a;
                        text-decoration: underline;
                     }
                    /*Active*/
                    a:active {
                        color: #88abc2;
                        font-weight: bold;
                    }

Radio Buttons

Los botones de opción (radio buttons) permiten a los usuarios seleccionar una opción de un conjunto mutuamente exclusivo. Solo se puede seleccionar una opción a la vez dentro del mismo grupo.

Código (HTML)

<label class="radio">
                    <input type="radio" name="tipo" value="estandar" checked>
                    <span class="check"></span>
                    <span>Estandar</span>
                    </label>

                        transition: .18s ease;
                        background: #fff;

                    .radio input {

                        inset: 0;

                    .radio .dot {

Checkboxes

Los checkboxes (casillas de verificación) son elementos de formulario que permiten al usuario seleccionar
una o varias opciones de una lista, cada una de las casillas funciona de manera independiente.
Ejemplos:

  1. Checkbox Normal:

    2. -Es el estado inicial por defecto.
    - El usuario puede marcarlo o desmarcarlo libremente.
    - Útil para opciones opcionales.

    Código en HTML:

    <label>
        <input type="checkbox" name="notificacion">
        Recibir notificaciones
    </label>

  2. Checkbox preseleccionado (checked)

    3. - Empieza marcado desde el inicio.
    - Se usa cuando quieres sugerir al usuario que esa opción ya está activada.
    - Aún puede desmarcarlo si quiere.

    Código en HTML:

    <label>
        <input type="checkbox" name="ofertas" checked>
        Recibir ofertas
    </label>

  3. Checkbox deshabilitado (disabled)

    4. -El usuario no lo puede modificar.
    -Se muestra en gris.
    -Se usa para opciones no disponibles o que dependen de otra condición.

    Código en HTML:

    <label>
        <input type="checkbox" name="actualizaciones" disabled>
        Recibir actualizaciones (deshabilitado)
    </label>

  4. Checkbox indeterminado (indeterminate)

    5. -Estado visual intermedio (ni marcado ni desmarcado).
    -Solo se puede activar con JavaScript.
    -Se usa en casos de selección múltiple (ej: "Seleccionar todos" cuando solo algunas opciones están marcadas).

    Código en HTML:

    <label>
        <input type="checkbox" id="miCheckbox">
        Selección parcial
    </label>
    <script>
        const chk = document.querySelector("#miCheckbox");
        chk.indeterminate = true;
    </script>

Select / Dropdown

Los elementos <select> en HTML se utilizan para crear menús desplegables (dropdowns). Estos permiten al usuario elegir una opción de una lista predefinida.
Cada opción se define con la etiqueta <option> dentro del <select>.

  1. Select básico:

    2. Este ejemplo muestra un menú desplegable con cuatro opciones.

    Código en HTML:

    <label for="frutas">Elige una fruta:</label>
    <select id="frutas" name="frutas">
      <option value="manzana">Manzana</option>
      <option value="banana">Banana</option>
      <option value="naranja">Naranja</option>
      <option value="uva">Uva</option>
    </select>

  2. Select con opción preseleccionada:

    3. La opción con el atributo selected aparece marcada por defecto.

    Código en HTML:

    <select id="pais" name="pais">
      <option value="cr">Costa Rica</option>
      <option value="mx" selected>México</option>
      <option value="es">España</option>
      <option value="ar">Argentina</option>
    </select>

  3. Select con múltiples opciones:

    4. Con el atributo multiple el usuario puede elegir más de una opción.

    Código en HTML:

    <select id="colores" name="colores" multiple size="4">
      <option value="rojo">Rojo</option>
      <option value="azul">Azul</option>
      <option value="verde">Verde</option>
      <option value="amarillo">Amarillo</option>
    </select>

Navbar

Es una barra de navegación que generalmente se ubica en la parte superior
de un sitio web y puede contener:
- Logo o nombre de la marca.
- Menú de navegación principal.
- Enlaces a las secciones más importantes.

Ejemplo de Barra de Navegación (Navbar):
Este nav tinene un logo a la izquierda y un menú de opciones a la derecha,
tiene un fondo semitransparente y un efecto de sombra para destacarse sobre el contenido.


LOGO

  • Servicios
