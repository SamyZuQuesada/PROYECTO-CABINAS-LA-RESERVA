Inputs

Los elementos <input> en HTML se usan para crear campos
campos interactivos en un formulario web, donde permite que los
usuarios ingresen y envien informacion de diferentes tipos, como texto, números,
correos electrónicos, etc.
Ejemplos:

Input de Texto :

Los elementos <input> de tipo text crean campos de texto básicos de una sola línea.

Código en HTML:

<input type="text" id="text" name="text" placeholder="Ingrese su texto aquí">

Input de Email :

Los elementos <input> de tipo email crean campos para ingresar direcciones de correo electrónico.

Código en HTML:

<input type="email" id="email" name="email" placeholder="Ingrese su correo electrónico">

Input de Password :

Los elementos <input> de tipo password crean campos para ingresar contraseñas de forma oculta.

Código en HTML:

<input type="password" id="password" name="password" placeholder="Ingrese su contraseña">

Input de Textarea :

Los elementos <textarea> se utilizan para crear campos de texto de varias líneas.

Código en HTML:

<textarea id="textarea" name="textarea" rows="4" cols="50" placeholder="Ingrese su texto aquí"></textarea>

Formularios

Los formularios en HTML se utilizan para recopilar información de los usuarios. Estos permiten enviar datos como texto, correos, contraseñas u opciones seleccionadas a un servidor o procesarlos en la misma página.
Ejemplos de uso: registro de usuarios, inicio de sesión, encuestas y formularios de contacto.

Formulario básico con estructura










Este ejemplo muestra un formulario con campos básicos para ingresar nombre, correo y mensaje.

Código en HTML:

<form>
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre" placeholder="Ingrese su nombre">

  <label for="correo">Correo electrónico:</label>
  <input type="email" id="correo" name="correo" placeholder="ejemplo@correo.com">

  <label for="mensaje">Mensaje:</label>
  <textarea id="mensaje" name="mensaje"></textarea>

  <button type="submit">Enviar</button>
</form>

Validación visual de formularios










En este ejemplo se aplican reglas de validación visual:
- El campo Usuario requiere mínimo 4 caracteres.
- El campo Correo electrónico debe tener un formato válido.
- La Contraseña requiere al menos 6 caracteres.
Si se intenta enviar el formulario sin cumplir los requisitos, el navegador muestra mensajes de error.

Código en HTML:

<form>
  <input type="text" id="usuario" name="usuario" required minlength="4">
  <input type="email" id="email2" name="email2" required>
  <input type="password" id="password2" name="password2" required minlength="6">
</form>

Links

Los links (enlaces) en HTML se crean con la etiqueta <a> y se pueden personalizar con
CSS para mostrar diferentes estados visuales.
Ejemplos:

Link Normal (a):

Es el estilo por defecto del enlace antes de que el usuario interactúe.

Hover (a:hover):

Se activa cuando el usuario pasa el cursor por encima del enlace.
Se suele usar para dar una pista visual (cambiar color, subrayar, aumentar tamaño).

Active (a:active):

Se activa cuando el usuario hace clic en el enlace y lo mantiene presionado.
Suele usarse para dar un efecto de "click real".

TOCA ESTE LINK

Código en HTML:

<a href="https://css-tricks.com/guides/">TOCA ESTE LINK</a>

Código en CSS: 

                    /*Normal*/
                    a{
                     color: #101652;
                     text-decoration: none;
                     font-size: 18px;
                    }
                    /*HOVER*/
                    a:hover {
                     color: #49708a;
                        text-decoration: underline;
                     }
                    /*Active*/
                    a:active {
                        color: #88abc2;
                        font-weight: bold;
                    }

Checkboxes

Los checkboxes (casillas de verificación) son elementos de formulario que permiten al usuario seleccionar
una o varias opciones de una lista, cada una de las casillas funciona de manera independiente.
Ejemplos:

Checkbox Normal:

-Es el estado inicial por defecto.
- El usuario puede marcarlo o desmarcarlo libremente.
- Útil para opciones opcionales.

Código en HTML:

<label>
    <input type="checkbox" name="notificacion">
    Recibir notificaciones
</label>

Checkbox preseleccionado (checked)

- Empieza marcado desde el inicio.
- Se usa cuando quieres sugerir al usuario que esa opción ya está activada.
- Aún puede desmarcarlo si quiere.

Código en HTML:

<label>
    <input type="checkbox" name="ofertas" checked>
    Recibir ofertas
</label>

Checkbox deshabilitado (disabled)

-El usuario no lo puede modificar.
-Se muestra en gris.
-Se usa para opciones no disponibles o que dependen de otra condición.

Código en HTML:

<label>
    <input type="checkbox" name="actualizaciones" disabled>
    Recibir actualizaciones (deshabilitado)
</label>

Checkbox indeterminado (indeterminate)

-Estado visual intermedio (ni marcado ni desmarcado).
-Solo se puede activar con JavaScript.
-Se usa en casos de selección múltiple (ej: "Seleccionar todos" cuando solo algunas opciones están marcadas).

Código en HTML:

<label>
    <input type="checkbox" id="miCheckbox">
    Selección parcial
</label>
<script>
    const chk = document.querySelector("#miCheckbox");
    chk.indeterminate = true;
</script>

Select / Dropdown

Los elementos <select> en HTML se utilizan para crear menús desplegables (dropdowns). Estos permiten al usuario elegir una opción de una lista predefinida.
Cada opción se define con la etiqueta <option> dentro del <select>.

Select básico:

Este ejemplo muestra un menú desplegable con cuatro opciones.

Código en HTML:

<label for="frutas">Elige una fruta:</label>
<select id="frutas" name="frutas">
  <option value="manzana">Manzana</option>
  <option value="banana">Banana</option>
  <option value="naranja">Naranja</option>
  <option value="uva">Uva</option>
</select>

Select con opción preseleccionada:

La opción con el atributo selected aparece marcada por defecto.

Código en HTML:

<select id="pais" name="pais">
  <option value="cr">Costa Rica</option>
  <option value="mx" selected>México</option>
  <option value="es">España</option>
  <option value="ar">Argentina</option>
</select>

Select con múltiples opciones:

Con el atributo multiple el usuario puede elegir más de una opción.

Código en HTML:

<select id="colores" name="colores" multiple size="4">
  <option value="rojo">Rojo</option>
  <option value="azul">Azul</option>
  <option value="verde">Verde</option>
  <option value="amarillo">Amarillo</option>
</select>
